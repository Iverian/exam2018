\section{Спектральный признак устойчивости разностной схемы Лакса для численного решения одномерного уравнения переноса}
\subsection*{Схема Лакса}
 Одномерное уравнение переноса:
$$\frac{dU}{dt} + \frac{dU}{dx} = f$$
Разностная схема
$$\frac{U_m^{n+1} - 1/2(U_{m+1}^{n} + U_{m-1}^{n})}{\tau}  + \frac{U_{m+1}^{n}  - U_{m-1}^{n}}{2h} = f_m^n$$
Порядок аппроксимирования: $O(\frac{h^2}{\tau} + \tau + h^2)$\\

\underline{Схема устойчива при $\frac{\tau}{h} \leq 1$}
